CSS 적용

  1. 인라인

    • HTML 태그의 style 속성으로 스타일을 직접 지정

    • 일반적인 실무 환경에서는 권장되지 않습니다.

  2. 내부 스타일

    • HTML의 head 요소 안에서, style 태그를 사용하여 스타일을 정의

    • HTML 파일 길이가 길어지기 때문에 효율적이지 않습니다.

  3. 외부 스타일

    • HTML 파일에 외부 CSS 파일을 연결하는 방식

    • link태그, rel="stylesheet", href="~/style.css" 로 연결

    • 다중 스타일 시트

      1. 기본 
        
    <!-- reset.css 브라우져의 스타일을 초기회 -->
    <link rel="stylesheet" href="./styles/reset.css">

    <!-- 폰트 -->
    <link rel="stylesheet" href="./styles/font.css">

    <!-- 레이아웃 -->
    <link rel="stylesheet" href="./styles/layout.css">
    
    <!-- reset.css의 내용을 덮어쓴다. -->
    <link rel="stylesheet" href="./styles/main.css">
